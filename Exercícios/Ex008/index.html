Citaçoes e Códigos

Se for necessário realizar uma citação a tag é Q

Se for nessário uma citação mais longa usa-se blockquote.

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Tenetur fugit cumque qui atque, animi consequatur? Odit soluta quis delectus rem quisquam, eos quam ipsum voluptates ratione necessitatibus eligendi modi dolores!

Para o uso de códigos usa a tag pre e code, no exemplo um código em python: 

def calculate():
        operation = input('''
    Please type in the math operation you would like to complete:
    + for addition
    - for subtraction
    * for multiplication
    / for division
    ''')
    
        number_1 = int(input('Please enter the first number: '))
        number_2 = int(input('Please enter the second number: '))
    
        if operation == '+':
            print('{} + {} = '.format(number_1, number_2))
            print(number_1 + number_2)
    
        elif operation == '-':
            print('{} - {} = '.format(number_1, number_2))
            print(number_1 - number_2)
    
        elif operation == '*':
            print('{} * {} = '.format(number_1, number_2))
            print(number_1 * number_2) calculate()
